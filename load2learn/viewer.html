Web Reader Help:

Menu Bar and Options

Below is a description of each icon on the toolbar starting from left to right.

Bookshare Web Reader logo Go back to RNIB UK Education Collection
Table of Contents Navigate the book by section or page number, as available within the book
Next and Previous Display the next or previous block of text
Play Begin text-to-speech reading from the top of what's currently displayed
Settings Modify layout, font, and audio settings
Night Mode Change the text to high contrast mode (black background with white text)
Full Screen Display the book in full screen, if supported by browser
Help Open this Help window
Hide Toolbar Hide the toolbar containing these buttons

Access Keys

If you are on Chrome and Internet Explorer:

  • Alt+P: Play/Stop text-to-speech (only available on later versions of Chrome and Safari)
  • Alt+B: Back to RNIB UK Education Collection
  • Alt+T: Toggle Table of Contents
  • Alt+N: Toggle Night Mode
  • Alt+F: Toggle Full Screen Mode
  • Alt+X: Hide Toolbar
  • Alt+V: Show Toolbar
  • Alt+O: Toggle Viewer Settings Dialog
  • Alt+1: Turn to Next Section
  • Alt+2: Turn to Previous Section

If you are on Safari:

  • Control+Option+P: Play/Stop text-to-speech (only available on later versions of Chrome and Safari)
  • Control+Option+B: Back to RNIB UK Education Collection
  • Control+Option+T: Toggle Table of Contents
  • Control+Option+N: Toggle Night Mode
  • Control+Option+F: Toggle Full Screen Mode
  • Control+Option+X: Hide Toolbar
  • Control+Option+V: Show Toolbar
  • Control+Option+O: Toggle Viewer Settings Dialog
  • Control+Option+1: Turn to Next Section
  • Control+Option+2: Turn to Previous Section

If you are on Firefox:

  • Alt+Shift+P: Play/Stop text-to-speech (only available on later versions of Chrome and Safari)
  • Alt+Shift+B: Back to RNIB UK Education Collection
  • Alt+Shift+T: Toggle Table of Contents
  • Alt+Shift+N: Toggle Night Mode
  • Alt+Shift+F: Toggle Full Screen Mode
  • Alt+Shift+X: Hide Toolbar
  • Alt+Shift+V: Show Toolbar
  • Alt+Shift+O: Toggle Viewer Settings Dialog
  • Alt+Shift+1: Turn to Next Section
  • Alt+Shift+2: Turn to Previous Section

Additional Reading Features Available

Bookshare Web Reader can let you hear words read aloud with Text-to-Speech and follow along with synchronized word highlighting. To enable these features, do one of the following:

• Update your Chrome browser to the latest version

OR

• If you can't update Chrome, click "Install Now" to install the Bookshare Web Reader browser extension.

Chrome Extension Available

To experience the full functionality of the Bookshare Web Reader, please install our Chrome Extension. With this extension, you will be able to:

  • See highlighting at the sentence and word level
  • Listen to the high-quality Text-to-Speech voice, synchronized with the text

Clicking "No Thanks" will prevent this message from appearing again, but you can still install the Bookshare Web Reader extension from the Chrome Web Store.
