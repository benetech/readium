Overview:

Bookshare Web Reader lets you read books directly from your Internet browser. The current version takes advantage of new and exciting features in the Google Chrome browser, which will allow you to read books multi-modally (see the text on the screen and hear it being read and highlighted), adjust font size, colors and display format, without additional reader software. The reader also works in Internet Explorer, Firefox, and Safari browsers, though those browsers do yet support the self-voicing functionality. For more information on compatible browsers, read below.

Requirements:

You must be an Individual Member and logged into Bookshare to use Bookshare Web Reader. If your session times out, you will need to re-authenticate using your Bookshare email and password.

Browsers:

Fully supported browsers with built in text-to-speech (TTS) and word-level highlighting:

  • Google Chrome version 23 (Google extension installation required)

Supported browsers for displaying books and reading with screen readers:

  • Mozilla Firefox version 17
  • Safari version 5.3
  • Internet Explorer version 9

*Please note that mobile browsers such as Safari for the Apple iPad, iPhone, or iPod Touch are not currently compatible.

Text-to-Speech (TTS) Voices:

The following voices can be used with the text-to-speech (TTS) and word-level highlighting option using Google Chrome:

  • Windows XP: Microsoft Mike or Mary
  • Windows Vista or 7: Microsoft Anna
  • Mac: Alex

Menu Bar and Options

Below is a description of each icon on the toolbar starting from left to right.

  • The Bookshare Web Reader logo allows you to go back to Bookshare.org.
  • The Table of Contents button allows you to navigate the book by Section or Page numbers. (Navigation features will depend on the book as some books will allow users to navigate by both section and page, or page only, or section only.) Page navigation will allow you to select a page from the drop down menu, or enter a specific page number.
  • The Next and Previous arrows will display the next or previous block of text.
  • The TTS (Text To Speech) button will initialize reading (Chrome browser with extension only). Please note the list of supported voices.
  • The Options button will modify your display settings.
    • You can change the font size text color and Display format here.
  • The Night Mode button will change the text to high contrast mode (black background with white text).
  • The Full Screen button displays the book in full screen.
    • This button appears only on browsers that support this capability.
  • The Hide Toolbar button will hide the toolbar.

Navigating books:

Next and Previous Page Buttons

To view the next section of text, either forward or backward in the book, select the next page/previous page buttons on the menu bar. Depending upon the font choice and the size of the browser window, this "next block of text" may not map exactly to a printed page.

Table of Contents

Another means of navigating the book is to use the Table of Contents. Select the Table of contents icon on the reader’s toolbar in order to open the table of contents and then navigate by chapter, section or page numbers, as defined in the original digital content.

  • Chapter titles are displayed as a list. To access a chapter please select a list and the reader will jump to the appropriate section.
  • Page numbers are displayed in a drop down menu. Once the drop down menu is selected, you'll be able to select a page number.
    • Only page numbers that are marked as such in the book are included in this dropdown menu; if a book has no titles, or is missing page numbers, that is because the book source is missing those page markings.

Go to Page

The Bookshare Web Reader allows navigation to a specific page number.

  • Open the Table of Contents and go to the drop down menu
  • Scroll through the list of page numbers and click, or type in the page number from your keyboard and then press the enter button, to jump to the appropriate page.

Changing Bookshare Web Reader Settings:

To access the Bookshare Web Reader display preferences, please select the options icon from the toolbar. A window will allow you to modify the display options for the reader. You’ll be able to change the font size, text and background color as well as the Display format.

Pagination Options

There are several types of display formats: single, facing, and scrolling page. The various options appear, based on whether your browser supports that capability.

  • Single page will show one page at a time and the user has to select the next page button to proceed to the next page (if you are using the Chrome extension, the reader will automatically move to the next page).
  • Facing will display 2 pages at a time and will function similar to the single page view.
  • Scrolling page will display the entire book on the page, allowing the user to scroll down to navigate the book.

Reporting Book Quality Issues

Bookshare wants to improve book quality. If you can help us by providing any book quality feedback, we'll work hard to make those changes and republish the books. Please submit a quality report by using the "Report book quality issue" link on the specific book information page.

Be sure to mention that you found the quality issue in a book you were reading with Bookshare Web Reader (vs. another format accessed on a different reading tool).
